# Guia-de-estudos-sobre-a-profissão-Social-Media
Neste repositório busco estudar quais  as habilidades necessárias para desenvolver e fazer parte da carreira de Social Media. Vamos entender exatamente quais as funções que um gestor das redes sociais fica responsável (Produção de postagens, conexão com a comunidade que usa ou interage com o produto envolvido), o que a legislação abrange sobre a profissão em 2026 e a realidade do mercado para um Social Media.
